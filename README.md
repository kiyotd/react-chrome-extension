# 参考

- [use create-react-app to develop Chrome extensions](https://dev.to/jamalx31/use-create-react-app-to-develop-a-chrome-extension-14ld)

# 動作確認

```bash
yarn install
yarn build
```
- Chrome の ```拡張機能を管理``` -> ```パッケージ化されていない拡張機能を読み込む``` から ```build``` ディレクトリを指定する。
